Ubuntu Tips

Jun 22, 2021

Disable animations, it makes the experience much much faster.

Once you disable animations, using the apps grid (shortcut SUPER + A by default) is lighting fast, and works as an application launcher (it covers the most basic functionality of applications like ulauncher).

Finally, try to set up a script to install your common programs, and also track every configuration change that you do. A git repo with a markdown file will do.

Many times reinstalling the operating system is the easiest option to do, and having that script and guide to set up your Ubuntu to your liking will be really useful. Needless to say the script doesn’t have to be perfect.
